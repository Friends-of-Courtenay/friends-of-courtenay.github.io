# 欢迎来到Courtenay之友！ { .center-title .courtenay-headline }
<div align="center" markdown>
![Courtenay Logo](logo.svg){width="120"}
</div>

位于Uptown的[Mary E. Courtenay语言艺术中心](https://courtenay.cps.edu/)提供从学前班到8年级的教育。如果您最近去过Courtenay或在附近，您一定见证了丰富的社区支持。然而，像许多社区学校一样，Courtenay资源不足。Courtenay之友旨在资助项目和材料，帮助教师和工作人员投资于每个孩子的巨大潜力和活力。

*Courtenay之友*是一个非营利组织，由照顾者、校友、社区成员和当地合作伙伴组成，他们投资于学校的学术[使命和愿景](https://courtenay.cps.edu/apps/pages/index.jsp?uREC_ID=412940&type=d)，以便Uptown的每一位年轻邻居都能拥有快乐的学习体验。

## Courtenay学生的食品安全保障

当SNAP福利被冻结时，邻居们团结起来帮助Courtenay家庭。我们在11月15日分发了**1,245美元的Aldi礼品卡**，并将在本周分发另外**1,000美元的食品援助**。

由于我们93%的学生来自低收入家庭，21%的学生住房不稳定，您的帮助至关重要。[^1] 我们的目标是在**寒假前再筹集1,000美元**。
