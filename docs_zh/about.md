# 关于Courtenay之友

Courtenay之友是一个非营利组织（501c3待定），由支持[Mary E. Courtenay语言艺术中心](https://courtenay.cps.edu/)使命和愿景的家长、照顾者、校友和社区成员组成。

## 我们如何使用捐款？

Courtenay之友完全由志愿者运营，没有受薪员工或管理费用。

目前，我们正在为紧急的*公平与基本需求*筹集资金，例如用于填补SNAP短缺的食品礼品卡，但我们的目标是建立可持续的资金，让Courtenay的学生能够像邻近学校的学生一样享受同样的丰富课程、艺术和课后机会。我们已经在与[当地学校委员会](https://www.courtenay.cps.edu/apps/pages/index.jsp?uREC_ID=412981&type=d)协商，以确定与Courtenay的[使命和愿景](https://courtenay.cps.edu/apps/pages/index.jsp?uREC_ID=412940&type=d)相一致的需求和支持机会。

## 社区支持的历史

![学生们正在绘制壁画](muralpaint.jpg "壁画和照片由@mollyz.art拍摄")

Courtenay的家庭和邻居在团结支持学校方面有着良好的记录。最近的一个例子是[人人皆宜的游乐空间](https://www.aplayspaceforall.com/)，旧的柏油路面被改造成了一个无论身体能力如何，每个孩子都能玩耍的空间。

我们将这个项目的成功视为社区长期需求和活力的证明，以支持学校建立包容性学习环境的愿景。
