# 关于 Courtenay 之友

Courtenay 之友是一个非营利组织（501(c)(3) 资格申请中），由支持 [Mary E. Courtenay 语言艺术中心](https://courtenay.cps.edu/) 使命和愿景的家长与监护人、校友和社区成员组成。

## 我们如何使用捐款？

Courtenay 之友完全由志愿者运营，没有受薪员工或管理费用。

目前，我们主要为紧急的 *公平与基本需求* 募集资金，例如用于弥补 SNAP 补助不足的杂货礼品卡；但我们的长期目标是建立可持续的资金，让 Courtenay 的学生也能像邻近学校的学生一样，享有丰富课程、艺术和课后项目等机会。我们已经开始与[当地学校委员会](https://www.courtenay.cps.edu/apps/pages/index.jsp?uREC_ID=412981&type=d)协作，识别与 Courtenay 的[使命和愿景](https://courtenay.cps.edu/apps/pages/index.jsp?uREC_ID=412940&type=d)相一致、最需要支持的方向与机会。

## 社区支持的历史

![学生们正在绘制壁画](muralpaint.jpg "壁画和照片由@mollyz.art拍摄")

Courtenay 的家庭与邻里社区长期以来一直团结支持学校。最近的一个例子是[人人皆宜的游乐空间](https://www.aplayspaceforall.com/)：原本的黑色沥青操场被改造成一个无论身体能力如何、每个孩子都能一起玩耍的空间。

我们将这个项目的成功视为社区长期需求和活力的证明，以支持学校建立包容性学习环境的愿景。
