# تبرع لأصدقاء كورتني 
يساعد تبرعك في تمويل الاحتياجات الأساسية، وموارد الفصول الدراسية، والرحلات الميدانية، والفعاليات العائلية، وتجارب أخرى لا يمكن للتمويل العام تغطيتها بمفرده.  

![رواق مشرق في كورتني](hallway.jpg "طلاب الصف الثالث في كورتني يمشون بهدوء في الرواق")

## للتبرع الآمن عبر الإنترنت

لدعم أصدقاء كورتني، ببساطة انقر على زر **ادعم اليوم** أدناه. تتم معالجة الهدايا عبر الإنترنت بشكل آمن من خلال شريك التبرع الخاص بنا.

[ادعم اليوم](https://19aid.com/courtenay-elementary-school-families-need-your-support/){ .md-button .md-button--primary }

## للتبرع بشيك أو أسهم

إذا كنت تفضل إرسال شيك بالبريد، أو التبرع بأسهم، أو لديك أفكار أخرى للتبرع، يرجى مراسلتنا عبر البريد الإلكتروني وسنقوم بمتابعة الأمر معك.

[راسل أصدقاء كورتني](mailto:FriendsOfCourtenay@proton.me?subject=Donation%20inquiry){ .md-button .md-button--secondary }
