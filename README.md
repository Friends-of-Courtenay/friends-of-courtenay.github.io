This is a website for (https://www.friendsofcourtenay.org/)[Friends of Courtenay], a 501(c)(3) non-profit organization that supports students, families, and staff at Mary E. Courtenay Language Arts Center.

It's built with [Zensical](https://zensical.org/), a modern static site generator (it's similar to mkdocs). We're happy to receive pull requests and contributions to the site.
