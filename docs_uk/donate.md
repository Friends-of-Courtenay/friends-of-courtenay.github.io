# Пожертвувати Друзям Кортні 
Ваш внесок допомагає фінансувати основні потреби, ресурси для класів, екскурсії, сімейні заходи та інші події, які державне фінансування саме по собі не може покрити.  

![Яскравий коридор у Кортні](hallway.jpg "Учні 3-го класу Кортні тихо йдуть коридором")

## Зробити безпечний внесок онлайн

Щоб підтримати Друзів Кортні, просто натисніть кнопку **Підтримати сьогодні** нижче. Онлайн-пожертви обробляються безпечно через нашого партнера.

[Підтримати сьогодні](https://19aid.com/courtenay-elementary-school-families-need-your-support/){ .md-button .md-button--primary }

## Зробити внесок чеком або акціями

Якщо ви бажаєте надіслати чек, передати акції або маєте інші ідеї щодо пожертви, будь ласка, напишіть нам, і ми зв'яжемося з вами.

[Написати Друзям Кортні](mailto:FriendsOfCourtenay@proton.me?subject=Donation%20inquiry){ .md-button .md-button--secondary }
