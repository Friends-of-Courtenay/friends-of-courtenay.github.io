# Dona a Amigos de Courtenay 
Tu donación ayuda a financiar necesidades básicas, recursos para el aula, excursiones, eventos familiares y otras experiencias que los fondos públicos por sí solos no pueden cubrir.  

![Un pasillo luminoso en Courtenay](hallway.jpg "Estudiantes de 3er grado de Courtenay recorriendo tranquilamente el pasillo")

## Para donar de forma segura en línea

Para apoyar a Amigos de Courtenay, simplemente haz clic en el botón **Apoya hoy** a continuación. Las donaciones en línea se procesan de forma segura a través de nuestra plataforma de donaciones.

[Apoya hoy](https://19aid.com/courtenay-elementary-school-families-need-your-support/){ .md-button .md-button--primary }

## Para donar con cheque o acciones

Si prefieres enviar un cheque por correo, donar acciones o tienes otras ideas para una donación, por favor, escríbenos y nos pondremos en contacto contigo.

[Envía un correo a Amigos de Courtenay](mailto:FriendsOfCourtenay@proton.me?subject=Consulta%20de%20donación){ .md-button .md-button--secondary }
